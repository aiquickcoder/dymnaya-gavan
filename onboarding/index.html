Что указать по каждому миксу
Чтобы карточка микса работала как витрина, для каждого нужны эти поля. Впишите свои миксы ниже — или пришлите списком менеджеру, оформим сами.
  1. Крепость — лёгкий / средний / крепкий
  2. Компоненты (табак) — бренд + вкус + %. Напр.: Musthave «Манго» 50% + Darkside «Банан» 50%
  3. Теги — «Сладкий», «Кислый», «Фруктовый»
  4. Описание — «Спелое манго с ноткой банана»
  5. Цена — можно разные для зон
  6. Категория — «Основное меню» или «Авторские миксы»
